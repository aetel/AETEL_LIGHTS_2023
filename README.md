# AETEL_LIGHTS_2023
This is how it works WLED AETEL lights.

Las luces de AETEL 2023 son básicamente tiras led WS2811B con las siguientes características:
- Funcionan a 5V.
- 60 LEDS/metro.
- Longitud 20 metros
- IP 30 Non WaterProof
- Color Negro
